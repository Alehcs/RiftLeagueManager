# LoL Esports — Private Data Pack

**For private prototype / friends use only. Not for public or commercial distribution.**

A private LoL esports catalog (v2): current **Tier 1** teams across all major
regions, **Tier 2 / academy** examples, and **historic / legacy / disbanded**
organizations for nostalgia templates. Team identities (name, short_name,
region) are real; rosters list verified handles only where known and auto-fill
the rest with clearly-flagged generated players — no invented names. No logos
are bundled (initials fallback).

➡️ **See [`TEAM_CATALOG.md`](./TEAM_CATALOG.md)** for the full team list by
section (current active, Tier 1, Tier 2/academy, historic / legacy / disbanded),
the competitions/templates, and all roster / logo / verification TODOs.

Reference: [Liquipedia Portal:Teams](https://liquipedia.net/leagueoflegends/Portal:Teams).
